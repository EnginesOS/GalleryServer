Overview

Engines is for running server applications on personal computers.

With Engines you can create powerful personal servers that can do any number of things for you. You can web apps, online stores, blogs, databases, media servers, file servers, game servers, backup servers and more. Use them to store and process your data, send websites to your browser, and be a back-end for apps on your smartphone.

People already use servers extensively in their everyday computing. Mostly they rely on the servers provided by big Internet companies. While this approach suits many, you may seek more control, lower costs, privacy and flexibility. You may want certainty of ownership over your software and the data it contains. If so, you need to run personal servers. The best way to run personal servers is to use Engines.

The technology is very cool

You run Engines on a computer at home, in the office, or you can use a virtual machine hosted in the cloud. The computer can be tiny or massively powerful.

Engines allows you to run multiple server applications on one computer. Each application can be focused on a different task, self-contained and executed separately for stability and security.

The architecture gives you the power to deploy multiple personal servers from inexpesnsive hardware. If a personal server should ever need more power, it is easy to add new physical or virtual hardware and supe-up Engines.

Engines is easy to use

With Engines, your server applications are quick and straightforward to install, configure and maintain.

The installation process is similar to the app store experience that you know from using your phone and tablet.

Use your browser to view applications in the software gallery. Select the server software that you require. The software will install and start. You now have a new personal server up and running.
